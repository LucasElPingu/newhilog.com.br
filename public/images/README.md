# Imagens do Projeto NEWHI Logística

Esta pasta contém todas as imagens estáticas do projeto.

## Arquivos necessários:

### `hero-truck.jpg`
- **Tamanho recomendado**: 1600x1000px
- **Formato**: JPG ou WebP
- **Descrição**: Imagem do caminhão na rodovia para o Hero section
- **Origem**: https://br.freepik.com/imagem-ia-gratis/caminhao-em-uma-rodovia-sob-um-ceu-ensolarado_419050559.htm
- **Status**: ⏳ Pendente - Você deve fazer download e colocar aqui

### `logo.png`
- **Tamanho recomendado**: 200x100px (ou maior para qualidade)
- **Formato**: PNG (com fundo transparente recomendado)
- **Descrição**: Logo da NEWHI Logística
- **Uso**: Header, Footer, Open Graph, Social Media
- **Status**: ⏳ Pendente - Você deve adicionar
